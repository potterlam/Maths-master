# Maths Master - Jedi Calculation Quest ⚔️🌟

A Star Wars–themed math runner game where you play as a Jedi running through space, answering calculation questions to upgrade your equipment and defeat evil bosses!

## 🎮 How to Play

1. **Your Jedi runs forward automatically** through 3 space lanes
2. **Math questions appear** at the top of the screen
3. **Move to the correct answer** using keyboard or touch:
   - **A** = Left lane | **S** = Center lane | **D** = Right lane
   - Or **tap/click** the lane on mobile
4. **Correct answers** earn points, build streaks, and unlock equipment!
5. **After 10 questions**, face the **Boss Fight**!
6. **Defeat the boss** to complete the level

## 📊 Levels

| Level | Operations | Name |
|-------|-----------|------|
| 1 | ＋ − | Padawan Training |
| 2 | × | Knight Trials |
| 3 | ÷ | Master Challenge |
| 4 | ＋ − × ÷ | Final Showdown |

## ⚔️ Equipment System

As you answer correctly, you unlock Jedi equipment:
- **3 correct** → First equipment piece
- **7 correct** → Second equipment piece

Each level has unique equipment to earn!

## 🎵 Features

- **Space-themed visuals** with animated starfield background
- **Web Audio API BGM** — ambient space music, boss battle theme, victory fanfare
- **Sound effects** for correct/wrong answers, equipment unlocks, boss hits
- **Streak system** with score multipliers
- **Boss fights** at the end of each level
- **Progress saved** in localStorage
- **Responsive** — works on desktop and mobile
- **Human-readable math symbols** (＋ − × ÷ instead of + - * /)

## 🚀 Quick Start

Simply open `index.html` in a web browser — no build tools or server required!

## 🛠 Tech Stack

- Pure HTML5, CSS3, JavaScript (ES6+)
- Canvas API for the runner game
- Web Audio API for procedural music and sound effects
- No external dependencies (except Google Fonts)
